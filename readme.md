# Magasin de pêche
## Démarrage
Assurez-vous d'avoir installé MariaDB.

Créez une base de donnée
```
CREATE DATABASE peche_magasin;
```
(modifier dans application.properties)

### (IntelliJ Idea)
Démarrer le projet: Maven -> `charger les dépendances`

Lancer le projet.

login : user

password : dans la console
